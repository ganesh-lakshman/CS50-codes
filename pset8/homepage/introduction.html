Sensors in vehicles

Introduction

An aspiration to make accident free travelling happen made us think of a project like this. Our project mainly focusses on introducing a new device to vehicles. The z generation has seen how devices can connect with each other. From generation to generation network speeds have been increasing exponentially. From downloading an mp3 file to playing a multiplayer game with 100 different users in the world we have came a long way. The world is racing towards 5G and millimeter waves, that made us think of this idea. What if our vehicles can communicate with each other? What if they can communicate with us. What if they took over our responsibility to turn on signal light whenever it is needed? Yes, today is the future. Our project revolves on an idea which makes all these things within the pocket full of bucks. We develop a sensor which is mounted Infront of the vehicle which do have a camera sensor with some other sensors. The camera sensor captures the roads we are travelling and gives input to the system we develop; the system automates the signal lights if the vehicle goes in the wrong way. It can be done using machine learning. Our machine learning model before the mass release will undergo many trainings and testing. This is just the beginning. There are no limits in imagining what this machine is capable of doing. We can control the device using a mobile application. And guess what we can unlock our vehicle with our phone.
